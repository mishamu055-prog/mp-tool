🔑 Token instelling
1. Open auctivo.net (ingelogd) → druk F12 → Network → filter op graphql
2. Refresh de pagina → klik een graphql request aan
3. Headers → kopieer de waarde na Authorization: Bearer
4. Plak hieronder en klik Opslaan — token blijft ~60 min geldig
⚡ Sneller: Sleep de knop hieronder naar je bookmarkbalk. Klik hem daarna op auctivo.net en het token wordt automatisch opgeslagen.
